## Prerequisites
* .NetCore 2.2

## Instructions
1. Clone and navigate to the repo.
2. Run ```dotnet restore``` if this is the first time running the application.
3. Run ```dotnet run```.
4. Enter ```https://localhost:5001``` into the URL of your browser.
5. Done!
